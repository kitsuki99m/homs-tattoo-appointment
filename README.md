
# 🖊️ HomS Tattoo Website

Personal website for **HomS Tattoo** — a tattoo studio based in Digos City, Davao del Sur, Philippines. This is a portfolio and booking site for tattoo artist **Jodex Wang**.

---

## 📖 About This Project

This is one of my early projects as someone who is still actively learning web development. I started with the basics and worked my way through building a real-world website for an actual client. It's been a great learning experience and I'm proud of how far it has come — even if there's still a lot more to learn.

> This project has no backend yet. I'm still in the process of learning server-side development, databases, and APIs. The booking form currently doesn't send or store data — that's coming in the future! 🚧

---

## ✨ Features

- 🖼️ Portfolio gallery
- 💰 Pricing guide
- 📅 Appointment booking form
- 📱 Responsive design for mobile and desktop
- 🎨 Smooth animations and transitions

---

## 🛠️ Built With

- ⚡ **Vite** — build tool and dev server
- 🎨 **Tailwind CSS** — utility-first CSS framework
- 🌸 **Flowbite** — Tailwind component library
- 🟨 **Vanilla JavaScript** — no frameworks, just plain JS
- 💻 **VS Code** — code editor

---

## 🤖 Development Notes

I'm still early in my web development journey. Throughout this project I used **AI-assisted tools** to help me understand concepts, debug issues, and improve my code. I believe using AI as a learning companion — not a shortcut — is a valid and modern way to grow as a developer.
